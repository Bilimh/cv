déc'Art

Description : déc'Art est un projet personnel que j'ai développé avec Django pour approfondir mes compétences en développement web tout en créant une plateforme de vente en ligne dédiée aux meubles de qualité. Cette boutique en ligne factice offre une expérience de shopping immersive et complète pour les amateurs de meubles.
Technologies utilisées : Pour ce projet, j'ai utilisé Django, un framework web Python puissant, pour gérer la logique de l'application et la gestion de la base de données. Le système de base de données repose sur PostgreSQL, offrant robustesse et performances. HTML, CSS, et JavaScript ont été employés pour créer l'interface utilisateur dynamique.
